# Coding
## コンポーネント
`.html`中に、
```
{ <コンポーネントパス(拡張子の.htmlは不要)> }
```
を入れると、対象の`.html`ファイルを検索して部品として配置する。

## 変数バインド
連想配列`BIND`内の要素は`.html`中で`{ :<変数名> }`でバインディングされる。  
例として、以下のコードでは`.html`中の`{:sugar}`のいちに`あまい`が入ってレンダリングされる。  
何かしらのイベントで変化した場合は再レンダリングされる。  
（下記コードでは、`sweet`から`あまい`へ再代入されたので、最新の`あまい`がレンダリングされる）
```
<script>
let BIND={
    sugar: "sweet"
}

BIND.sugar="あまい"
</script>

<body>
<p>{:sugar}</p>
</body>
```
