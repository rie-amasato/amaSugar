# あま砂糖
Javascript、HTML向けのSyntaxSugar

# Coding
## コンポーネント
`.html`中に、
```
{ <コンポーネントパス(拡張子の.htmlは不要)> }
```
を入れると、対象の`.html`ファイルを検索して部品として配置する。

## 入力コンポーネント
テキスト入力コンポーネントは`{input :in}`のようにすると`BIND`内の変数をコンポーネント内外で共有できる  
詳細は入力コンポーネント制作の項を参照

## 変数バインド
連想配列`BIND`内の要素は`.html`中で`{ :<変数名> }`でバインディングされる。  
例として、以下のコードでは`.html`中の`{:sugar}`のいちに`あまい`が入ってレンダリングされる。  
何かしらのイベントで変化した場合は再レンダリングされる。  
（下記コードでは、`sweet`から`あまい`へ再代入されたので、最新の`あまい`がレンダリングされる）
```
<script>
let BIND={
    sugar: "sweet"
}

BIND.sugar="あまい"
</script>

<body>
<p>{:sugar}</p>
</body>
```

# 入力コンポーネント作成
`{<コンポーネント名> :<BIND可能変数>}`のように、コンポーネントの内外で共有する変数を`:`で分けて記述する  

変数の番号が0から始まり、`$in_<番号>`として入力コンポーネント内で記述する  
入力コンポーネント内の`.html`中でその変数をバインドしたいとき`{:$in_0}`  
入力コンポーネント内のスクリプト中でその変数を使用したいとき`$$in_0`  
のように記述する
